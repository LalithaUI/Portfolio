NetBeans

We use NetBeans 5.5 and 5.0 as an IDE (Integrated Development Environment) for Java, j2EE and J2ME application development. NetBeans supports core java application, applet based application and SWT application development.

When we consider web projects, We have found that NetBeans is ideally preferred for application development as it provides tomcat and jboss as application servers. We need just choose which application server we wish to use while executing the web project. As part of developmental activities we use Breakpoints for debugging the application.

This is yet another very welcome feature of NetBeans. Also the fact that we can connect to the Database through the Netbeans IDE which has a wonderful feature that allows us to drag and drop database into our application generating the database related code. The NetBeans IDE allows us to execute queries directly.

When we use NetBeans for J2ME application development, we have to configure the J2ME wireless toolkit with Netbeans through the Netbeans Mobility pack provided as a plug-in. Once this is done we are able to execute the application using emulators.
